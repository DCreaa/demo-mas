# Research Paper Analysis Multi-Agent System Demo

A complete demonstration of Multi-Agent Systems using LangGraph and Ollama.

## Installation Guide

1. Download and Install Ollama:
   ```
   https://ollama.com/download/
   ```
2. Upgrade pip:
   ```shell
   python -m pip install --upgrade pip
   ```
3. Install all packages:
   ```shell
   pip install -r requirements.txt
   ```

## Running the Demo

1. Ensure Ollama is running

3. Run the demo with sample paper:
   ```shell
   python main.py
   ```
